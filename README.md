# PopClip Extensions

我的 PopClip 插件集合。  
My collection of PopClip extensions.

## Extensions / 插件列表

- [IPLookup.popclipext](./IPLookup.popclipext): Look up basic information about IPv4 and IPv6 addresses. / 查询 IPv4 和 IPv6 地址的基本信息。

## Directory Publishing / 目录发布

This repository is structured for publishing extensions from a personal GitHub repository.  
这个仓库的结构适合从个人 GitHub 仓库发布 PopClip 插件。

Current directory include path / 当前目录 include 路径:

- `include=IPLookup.popclipext`

## Notes / 说明

- Source directories are committed to Git. / Git 中提交的是源码目录。
- Packaged `.popclipextz` files are excluded and can be attached to GitHub Releases if needed. / 打包产物 `.popclipextz` 不纳入版本库，必要时可作为 GitHub Release 附件上传。
