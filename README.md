## English

My collection of PopClip extensions.

### Extensions

- [IPLookup.popclipext](./IPLookup.popclipext): Look up basic information about IPv4 and IPv6 addresses.

### Directory Publishing

This repository is structured for publishing extensions from a personal GitHub repository.

Current directory `include` path:

- `include=IPLookup.popclipext`

### Notes

- Source directories are committed to Git.
- Packaged `.popclipextz` files are excluded and can be attached to GitHub Releases if needed.

### Changelog

- 2026-03-08: Added the first extension, `IPLookup.popclipext`.
